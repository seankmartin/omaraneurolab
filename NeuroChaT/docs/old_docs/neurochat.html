class neurochat.nc_config.Configuration(filename=[])¶

Bases: object

The Configuration object is the placeholder for all the settings of NeuroChaT consisting of specification of data and analysis along with the parameters for each analysis type.

It also facilitates saving these setting to a .ncfg file and retrive them from the file. The .ncfg file is a YAML-formatted file.
